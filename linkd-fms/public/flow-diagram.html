3. Role Permissions

Admin Full Power

  • All Tasks, Task Dashboard, Concept Dashboard
  • Concept approval (review/reject/revision)
  • Change user roles + manage designer codes
  • Manage lookup tables (System CRUD)
  • CSV Export on all views
  • Sampling, Salvedge, Scorecards, Team

Coordinator Almost Admin

  • Same as Admin (permissions now equal)
  • Concept approval + review
  • Task lifecycle (create, assign, advance)
  • System CRUD access
  • Sampling, Salvedge, Team

Designer Submit & Work

  • My Board, Task Dashboard, Concepts
  • Claim tasks from pool
  • Submit + finalize concepts
  • Salvedge view
  • Cannot create briefs or manage team

5. Concept Approval Workflow

Designer submits concept
Title + file + client + priority · DB auto-generates code
↓
Admin reviews within 1 day
ConceptDetailDrawer → approve / reject / revision
↓
Approve
Designer has +4 days to finalize
↓
Final Approval
Admin grants final approval
Revision
Designer fixes + re-submits
Reject
Terminal state
Monthly target: 3 approved concepts per designer
Notifications: Concept submitted → admins, reviewed → designer, overdue concepts → alerts
Dashboards: Concept Dashboard (funnel + leaderboard + target race + MD review panel)
