# LinkD Design Flow System

Textile design workflow + file management for **LinkD Prints**.

## Project structure

```
linkd-fms/          Active app (Vite + React + TypeScript + Tailwind + Supabase)
scripts/            Migration & seed scripts (Node.js)
supabase/           Database migrations (SQL)
CLAUDE.md           Full codebase documentation
```

## Getting started

```bash
cd linkd-fms
npm install
npm run dev          # http://localhost:5173
```

See [CLAUDE.md](CLAUDE.md) for full documentation.
